flexbox 레이아웃 설명

flex 란?

flex는 더 나은 정렬, 정렬 및 공간 배분 기능을 제공하는 css 레이아웃 모듈입니다. 주로 한 방향(가로 또는 세로)으로 아이템을 배치하는데 사용합니다.

기본 사용법

기본 flexbox 레이아웃

아이템1
아이템2
아이템3
.flex-container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

주요 속성

flex-direction

flex-direction속성은 아이템 정렬 방향을 설정합니다.

아이템1
아이템2
아이템3
.flex-container {
  flex-direction: column;
}

justify-content

justify-content 속성은 주축에서 아이템의 정렬 방식을 설정합니다.

아이템1
아이템2
아이템3
.flex-container {
  justify-content: space-around;
}

align-items

align-items 속성은 교차축 에서 아이템의 정렬 방식을 설정합니다.

아이템1
아이템2
아이템3
.flex-container {
  align-items: flex-end;
  height: 200px;
}

flex-wrap

flex-wrap 속성은 아이템이 컨테이너를 벗어날 때 줄 바꿈을 할지 여부를 설정합니다.

아이템1
아이템2
아이템3
아이템4

align-content

align-content 속성은 여러줄의 아이템을 (교차축)정렬하는 방식을 설정합니다.

아이템1
아이템2
아이템3
아이템4
아이템5
아이템6
.flex-container {
  flex-wrap: wrap;
  align-content: space-between;
  height: 400px;
}
.flex-item {
  flex: 0 0 45%;
}

flex 아이템 속성

flex-grow

flex-grow 속성은 아이템이 남은 공간을 어떻게 나누어 가질지를 설정합니다.

아이템1
아이템2
아이템3
.flex-item {
  flex-grow: 1;
}
.flex-item:nth-child(2) {
  flex-grow: 2;
}

flex-shrink

flex-shrink 속성은 아이템이 축소될 때 어떻게 줄어들지를 설정합니다.

아이템1
아이템2
아이템3
.flex-container {
  width: 200px;
}
.flex-item {
  flex-shrink: 1;
}
.flex-item:nth-child(1) {
  flex-shrink: 0;
} .flex-item:nth-child(3) {
  flex-shrink: 2;
}

flex-basis

flex-basis 속성은 아이템의 기본 크기를 설정합니다.

아이템1
아이템2
아이템3
.flex-item {
  flex-basis: 100px;
}
.flex-item:nth-child(2) {
  flex-basis: 200px;
}
